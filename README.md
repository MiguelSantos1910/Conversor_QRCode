# Gerador de QR Code

Um projeto simples em **HTML, CSS e JavaScript** que permite gerar QR Codes a partir de qualquer link ou texto. Ideal para uso rápido e fácil na web.  

---

## Funcionalidades

- Gera QR Code a partir de qualquer link ou texto.
- QR Code centralizado e estilizado.
- Validação para evitar QR Codes vazios.
- Interface moderna e responsiva.
- Permite gerar múltiplos QR Codes (limpa o anterior ao gerar outro).

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura da página.
- **CSS3** – Estilização moderna e responsiva.
- **JavaScript** – Lógica de geração do QR Code.
- **[QRCode.js](https://davidshimjs.github.io/qrcodejs/)** – Biblioteca para gerar QR Codes.
